# ແອັບແກ້ໄຂປ້າຍລາຄາຄຳ v2

ເວັບແອັບແບບ local ທີ່ສາມາດຍ້າຍໂຟນເດີໄປໃຊ້ຕໍ່ໄດ້

## ວິທີຮັນ

1. ດັບເບິນຄລິກ `start.bat`
2. browser ຈະເປີດທີ່ `http://127.0.0.1:3210`

## ຄວາມສາມາດ

- ໃຊ້ template ຈາກໂຟນເດີນີ້ໂດຍກົງ
- ກົດ `ອັບໂຫຼດ Template ໃໝ່` ເພື່ອແທນ template ເກົ່າ
- ເກັບ template ທີ່ໃຊ້ງານໄວ້ພຽງ 1 ໄຟລ໌ໃນ `assets/`
- ສະແດງ preview ຢູ່ດ້ານຂວາ
- ລາກກ່ອງເທິງ preview ເພື່ອຂະຍັບຕຳແໜ່ງຂໍ້ຄວາມ
- ກົດ `ບັນທຶກ Layout` ເພື່ອບັນທຶກຕຳແໜ່ງລົງ `data/layout.json`
- ກົດ `ບັນທຶກຕຳແໜ່ງນີ້ເປັນ Default ໃໝ່` ເພື່ອບັນທຶກຄ່າຕັ້ງຕົ້ນໃໝ່ລົງ `data/default-layout.json`
- ກົດ `Export PNG ສຸດທ້າຍ` ເພື່ອສ້າງຮູບສຸດທ້າຍ

## ການຍ້າຍໄປເຄື່ອງອື່ນ

ສາມາດຄັດລອກທັງໂຟນເດີນີ້ໄປໃຊ້ຕໍ່ໄດ້ເລີຍ ໂດຍເຄື່ອງປາຍທາງຕ້ອງມີ Node.js ສຳລັບຮັນ `start.bat`
